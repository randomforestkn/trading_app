# Release Checklist

Use this checklist before shipping a demo or store build.

## Store assets
- App icon
- Splash screen
- App screenshots for phone and tablet
- Store description and feature summary

## Legal and policy
- Privacy policy page or placeholder replaced with a real URL before release
- Terms and disclaimer screen reviewed
- No real-money trading statement
- No investment advice statement
- Options risk disclaimer reviewed

## Product behavior
- Paper trading mode confirmed as default
- Remote market data remains optional
- Local export and restore behavior verified
- Demo auth remains local only
- Journal, analytics, insights, strategy simulator, and options portfolio render correctly

## Data safety
- Data storage answers documented
- Local backup and restore behavior explained
- Import/restore warning copy reviewed
- No secrets or hardcoded API keys in source control

## Demo and QA
- Demo account instructions documented
- Reset, clear history, export, and restore flows tested
- Compact mobile layout verified
- Tablet and desktop width smoke tests verified
- Startup loading and restore flow verified

## Build notes
- Android release build checked
- iOS release build checked
- Version label reviewed
- Build mode label reviewed
